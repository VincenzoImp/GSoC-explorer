This guide will help you compile [VLC media player](https://wiki.videolan.org/VLC_media_player/) for your platform.

This guide is intended for **developers** and **power users**. Compiling VLC is not an easy task.

You should come on [IRC](https://wiki.videolan.org/IRC/) to assist you during this phase.

**Warning:** Please, do not put pre-release test binaries or git-compiles on software sites or on user-forums. We've had bad experiences with this before and we do not appreciate it.

## Generic Steps

These are the main steps to follow to compile VLC :

- Prepare your compile environment,
- Grab the
[source](https://wiki.videolan.org/GetTheSource/)of[VLC media player](https://wiki.videolan.org/VLC_media_player/), - Bootstrap, using the ./bootstrap script, if you took the code from repository,
- Prepare the extras libraries needed for
[VLC media player](https://wiki.videolan.org/VLC_media_player/), read the[list](https://wiki.videolan.org/Contrib_Status/), [Configure](https://wiki.videolan.org/Configure/),- Build, using make,
- and Profit!
