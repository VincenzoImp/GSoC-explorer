There are several ways to contribute code to ns-3. The options are outlined in our [contributing guide](https://www.nsnam.org/docs/contributing/html/index.html).

- Contribute to the main tree through our code contribution process (outlined in the above linked documentation)
- Maintaining your own ns-3 extension as a contributed module in our
[ns-3 App Store](https://apps.nsnam.org) - Archive some software either at your own website or a public Git repository, and link to it from this page

## Contributed code under review for the main tree

Contributed code for the main tree will be accepted if it is considered to be of general interest. Many new contributions for less common (or more niche) use cases are recommended for the ns-3 [App Store](https://apps.nsnam.org/). View current [Merge Requests](https://gitlab.com/nsnam/ns-3-dev/-/merge_requests) for the mainline ns-3-dev. See the [ Current Development page](https://www.nsnam.org/wiki/Current_Development#Code_reviews) and [code review discussions](https://groups.google.com/forum/?fromgroups#!forum/ns-3-reviews) for a listing of older code that may still be of interest.
