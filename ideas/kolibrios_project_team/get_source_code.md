# Get source code

[Jump to navigation](https://wiki.kolibrios.org#mw-head)

[Jump to search](https://wiki.kolibrios.org#searchInput)

The KolibriOS developer team makes use of a central Git source code repository. It contains not only KolibriOS kernel and drivers, but also almost all programs and libraries that are present in this OS.

The repository is located [https://git.kolibrios.org/KolibriOS/kolibrios.git](https://git.kolibrios.org/KolibriOS/kolibrios.git)


There are also read-only repository mirrors:

Keep in mind that these repositories are for checkouts only, not commits.

## Installing Git

### Linux

- Ubuntu

`apt-get install git`


- Arch Linux

`pacman -S git`

- Mountain Lion

## Git GUI

If you find it more comfortable to work with Git through GUI, rather than a terminal, you can use the built-in git-gui or pick one of many GUI clients listed on the official website:

In particular:

[TortoiseGit](https://tortoisegit.org/)for Windows[gitg](https://gitlab.gnome.org/GNOME/gitg/)for Linux[Gitnuro](https://gitnuro.com/)for MacOS

Alternatively, you can use Git extensions for your favorite code editor, like [Source control](https://code.visualstudio.com/docs/sourcecontrol/overview) in VSCode.
