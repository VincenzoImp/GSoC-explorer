# GNU Mailman

> GNU Mailman manages your mailing lists.

**Technologies:** python, django, rest, sqlalchemy, zope
**Topics:** email, archives, list management
**Website:** https://www.list.org
**Ideas:** https://wiki.list.org/DEV/Google%20Summer%20of%20Code%202026
**GSoC Page:** https://summerofcode.withgoogle.com/programs/2026/organizations/gnu-mailman

## Description

The Mailman project develops the GNU Mailman mailing list manager. Our goals are RFC conformance and powerful convenient tools for subscribers, moderators, list owners, domain administrators, and site adminstrators of mailing lists.
