# OpenWISP

> The Hackable Network Management System

**Technologies:** python, javascript, django, lua, openwrt
**Topics:** networking, network management system, wifi, vpn, sdn
**Website:** https://openwisp.org
**Ideas:** https://openwisp.io/docs/dev/developer/gsoc-ideas-2026.html
**GSoC Page:** https://summerofcode.withgoogle.com/programs/2026/organizations/openwisp

## Description

OpenWISP is an open source network management system which runs on low cost hardware and can be used to manage networks: from public wifi, university wifi, to mesh networks and IoT.
