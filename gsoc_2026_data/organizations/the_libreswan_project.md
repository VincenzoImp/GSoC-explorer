# The Libreswan Project

> Encrypting the Internet with IKE / IPsec VPN

**Technologies:** c, kernel, nss, RFCs, libevent
**Topics:** vpn, ipsec, ikev2
**Website:** https://libreswan.org
**Ideas:** https://github.com/libreswan/libreswan/wiki/GSoC-2026-Code-Project-Ideas
**GSoC Page:** https://summerofcode.withgoogle.com/programs/2026/organizations/the-libreswan-project

## Description

Libreswan implements the IKE and IPsec standards for VPN.  These standards have been created and are still maintained at the Internet Engineering Task Force (IETF) in the IPsecME Working Group.  Libreswan is used as a remote access VPN as well as site-to-site and cloud encryption.
