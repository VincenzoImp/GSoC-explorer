# MIT App Inventor

> Anyone can build apps with global impact

**Technologies:** javascript, java, gwt, swift
**Topics:** education, development tools, android, ios
**Website:** http://appinventor.mit.edu
**Ideas:** https://github.com/mit-cml/appinventor-sources/wiki/Google-Summer-of-Code-2026
**GSoC Page:** https://summerofcode.withgoogle.com/programs/2026/organizations/mit-app-inventor

## Description

MIT App Inventor is a free, open source web app that anyone can use to build mobile apps. It has been used by over 8 million people worldwide who have built more than 30 million apps. It is available in twelve different languages and used by people from ages 13 and up.
