## Description

API Dash is a beautiful open-source cross-platform (macOS, Windows, Linux, Android & iOS) API Client built using Flutter & powered by AI which can help you easily create & customize your HTTP, GraphQL, SSE & AI API requests, visually inspect responses and generate API integration code.

This season we are focussing on helping developers test MCP tools/servers, evaluating Multimodal AI APIs, and adding agentic AI features in API Dash.

Project Tech stacks - React, Node, TypeScript, Flutter, Dart & Python.
