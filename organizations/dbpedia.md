# DBpedia

> Global and Unified Access to Knowledge Graphs.

**Technologies:** python, javascript, java, scala, rdf
**Topics:** machine learning, semantic web, linked data, knowledge graph, largelanguagemodel
**Website:** https://www.dbpedia.org/
**Ideas:** https://forum.dbpedia.org/tag/gsoc2026-ideas
**GSoC Page:** https://summerofcode.withgoogle.com/programs/2026/organizations/dbpedia

## Description

DBpedia is a crowd-sourced community effort to extract structured information from Wikipedia and make this information available on the Web. It allows for a global and unified access to Knowledge Graphs.
