## Description

JBoss Community is a community of open source projects. The community hosts a diverse set of projects that are written in various programming languages. The primary language is Java, however there are also projects that are written in Go, Rust, Ruby, PHP, Node and other languages.

Project categories range from application servers, microservices, IOT, cloud technologies, web frameworks et cetera
