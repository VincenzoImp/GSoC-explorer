## Description

Open Robotics supports the development, distribution, and adoption of open source software for use in robotics research, education, and product development. Open Robotics is an umbrella organization encompassing five projects: 
- Robot Operating System (ROS), a robot development framework.
- Gazebo, a robot simulation framework.
- Open-RMF, a framework for coordinating multiple fleets of robots.
- ros-controls, an extensive control framework built for ROS.
- The ROS Infrastructure that is used to build and distribute ROS packages.
