## How this data was collected

A Python scraper queries the GSoC public API, then fetches each org's ideas page:

- GitHub blob/wiki/gist URLs are converted to raw URLs for clean Markdown
- Google Docs are exported as HTML/text
- GitHub/GitLab issues are fetched via CLI/API
- HTML pages are extracted via trafilatura or markdownify
- Playwright headless browser handles JS-rendered pages

See [`scraper/`](scraper/) for the source code.
